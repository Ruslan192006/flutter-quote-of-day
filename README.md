# Flutter Quote of the Day

Приложение с вдохновляющими цитатами дня.

## Функциональность
- Случайная цитата дня
- Избранные цитаты
- Категории цитат
- Поделиться цитатой
- Красивый UI

## Технологии
- Flutter
- SharedPreferences
- Material Design
- Animations

## Запуск
```bash
flutter pub get
flutter run
```
